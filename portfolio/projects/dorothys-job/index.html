Dorothy’s Job

UE5 C++ FMOD UI Programming

About the Project

Dorothy’s Job is a frenetic and comedic isometric twin-stick shooter where players control a maid fighting through chaotic enemy-filled levels.

This page highlights my technical contributions, focused on UI systems, audio programming, and gameplay integration in Unreal Engine 5.

My Role

UI Programming

  • Custom C++ focus navigation system (grid + linear UI)
  • Input-aware navigation (keyboard/gamepad/mouse)
  • Focus wrapping + focus memory system
  • Responsive UMG layouts across resolutions

Audio Programming

  • FMOD integration with event-driven architecture
  • Custom Audio Manager subsystem
  • Audio settings management system

Gameplay Contributions

  • Modular consumable / power-up system
  • Weapon and enemy behavior support

Key Systems

Focus Navigation System

Modular UI navigation system supporting both grid and linear layouts with input-aware control and focus memory.

Audio Manager (FMOD)

Centralized subsystem handling all audio events and settings across gameplay systems.

Consumable System

Scalable architecture for power-ups integrated into player gameplay.
